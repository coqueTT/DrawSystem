# DrawSystem

## 操作説明
幾何的図形が描けるドローソフトです。

図形を描く際の初期設定は左バナーから。
選択モードで選択した図形に対する変更は右クリックまたは上部バナーから適用して下さい。

色の初期設定は
色を選択→線・塗り・影のボタンを押す
の順で適用したい部分の色を変更して下さい

図形のリサイズは■をドラックすると出来ます。

## 現在確認中の不具合
線の太さ・重線の値を入力した後、ENTERを押さなければ適用されない。

ハートに破線が適用されない。
